## 📋 使用方式

### 批量上傳腳本 (`批量上傳-修正版.bat`)

1. **第一次使用**：
   - 執行腳本時會提示輸入設定
   - 直接按 Enter 會使用預設值
   - 設定會儲存到 `上傳設定.txt` 檔案中

2. **之後使用**：
   - 如果存在 `上傳設定.txt`，會自動載入設定
   - 可以選擇使用現有設定或重新設定

3. **修改設定**：
   - 刪除 `上傳設定.txt` 檔案，下次執行時會重新詢問
   - 或選擇「不使用現有設定」來重新設定

### 單網站上傳腳本 (`上傳網站.bat`)

- Strapi URL 和 Token 已預設在腳本中
- 如果需要修改，可以編輯 `上傳網站.bat` 檔案

## 🔧 如何修改預設值

### 方法 1：修改設定檔
編輯 `獨立腳本包/上傳設定.txt`：
```
GITHUB_URL=https://github.com/your-username/your-repo
GITHUB_BRANCH=main
STRAPI_URL=https://your-strapi-url.strapiapp.com
STRAPI_TOKEN=your-token-here
```

### 方法 2：修改腳本檔案
編輯 `獨立腳本包/批量上傳-修正版.bat`，找到預設值並修改：
```batch
set GITHUB_URL=https://github.com/your-username/your-repo
set STRAPI_URL=https://your-strapi-url.strapiapp.com
```

## 💡 提示

- 預設值已經設定好，大部分情況下直接按 Enter 即可
- 如果需要使用不同的倉庫或 Strapi 實例，可以隨時修改
- 設定檔會自動儲存，下次使用時不需要重新輸入
