# 開發環境網站訪問說明

## 如何查看開發環境的網站

### 方法一：透過 Vercel Deployments 頁面（最簡單）

1. **登入 Vercel**
   - 前往 https://vercel.com
   - 登入你的帳號

2. **進入專案**
   - 找到專案：`multi-site-static-strapi-front`
   - 點擊進入專案

3. **查看 Deployments**
   - 點擊頂部選單的 **Deployments**
   - 找到 `develop` 分支的部署記錄
   - 點擊該部署記錄
   - 點擊右上角的 **Visit** 按鈕

### 方法二：使用預覽網址

每個部署都會有一個唯一的預覽網址：

1. 在 Deployments 頁面找到 `develop` 分支的部署
2. 部署記錄會顯示預覽網址，格式類似：
   ```
   https://multi-site-static-strapi-front-xxxxx.vercel.app
   ```
3. 點擊該網址即可訪問

### 方法三：使用環境網址

如果設定了不同的環境：

- **Production 環境**（develop 分支）：
  ```
  https://multi-site-static-strapi-front.vercel.app
  ```

- **Preview 環境**（其他分支）：
  ```
  https://multi-site-static-strapi-front-git-分支名-你的帳號.vercel.app
  ```

## 多站點訪問路徑

由於你的專案是多站點結構，訪問路徑如下：

### 開發環境（develop 分支）
```
https://你的域名.vercel.app/site1/index.html
https://你的域名.vercel.app/site2/index.html
https://你的域名.vercel.app/site3/index.html
...
https://你的域名.vercel.app/site16/index.html
```

### 範例
如果 Production 環境使用 develop 分支：
```
https://multi-site-static-strapi-front.vercel.app/site1/index.html
https://multi-site-static-strapi-front.vercel.app/site4/index.html
```

## 如何找到確切的網址

1. **在 Vercel 專案中**
   - 點擊 **Deployments**
   - 找到最新的 `develop` 分支部署
   - 點擊部署記錄
   - 在頁面頂部會顯示網址

2. **在專案設定中**
   - 點擊 **Settings** → **Domains**
   - 可以看到所有設定的網域
