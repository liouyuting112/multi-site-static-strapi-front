## 快速查找方法

### 方法一：從 Vercel 部署頁面

1. **點擊 Visit 按鈕**
   - 在部署詳情頁面，點擊右上角的 **Visit** 按鈕
   - 這會帶你到根網址

2. **手動加上路徑**
   - 在網址後面加上 `/site1/index.html`
   - 或 `/site2/index.html` 等

### 方法二：直接訪問

1. **複製基礎網址**
   - 從部署頁面的 **Domains** 區塊複製網址

2. **加上站點路徑**
   - 格式：`基礎網址/siteX/index.html`
   - 例如：`https://xxx.vercel.app/site4/index.html`

### 方法三：使用 Vercel 的網址結構

Vercel 會自動處理路徑，你也可以嘗試：

```
https://基礎網址/site1/
https://基礎網址/site2/
https://基礎網址/site4/
```

（Vercel 通常會自動重定向到 `index.html`）

## 注意事項

⚠️ **如果看到 404 錯誤：**

1. **確認路徑正確**
   - 確保使用 `/siteX/index.html` 格式
   - 不是 `/siteX/`（雖然通常可以，但有時需要完整路徑）

2. **確認站點存在**
   - 檢查 GitHub 分支中是否有對應的 `siteX` 資料夾
   - 確認 `siteX/index.html` 檔案存在

3. **檢查部署狀態**
   - 確認部署狀態是 **Ready**（綠色）
   - 如果還在 Building，需要等待完成
