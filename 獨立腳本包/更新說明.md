# 🔄 更新說明

## ✨ 新版本特色

### 支援任何網站名稱

**舊版本限制：**
- ❌ 只支援 site6-10
- ❌ 網站名稱必須符合特定格式

**新版本優勢：**
- ✅ 支援任何網站名稱
- ✅ site6, site11, site20, my-website, 任何名稱都可以
- ✅ 自動從資料夾名稱提取網站名稱

---

## 📋 檔案變更

### 主要變更

1. **`upload-site-to-strapi.js`**（新檔案）
   - 通用版本，支援任何網站名稱
   - 自動從資料夾名稱提取網站名稱

2. **`upload-site6-10-to-strapi.js`**（舊檔案）
   - 保留作為備用
   - 腳本會自動尋找新版本

3. **`上傳網站.bat`**（已更新）
   - 優先使用 `upload-site-to-strapi.js`
   - 如果找不到，會自動使用舊版本

---

## 🚀 使用方式

### 完全一樣，沒有任何改變！

**步驟 1：下載腳本包**
```
上傳網站.bat
upload-site-to-strapi.js  ← 新版本（推薦）
```

**步驟 2：準備網站資料夾**
```
桌面/
└── site11/              ← 可以是任何名稱！
    ├── index.html
    └── articles/
        └── 2025-12-01.html
```

**步驟 3：執行腳本**
```bash
上傳網站.bat
```

**輸入網站資料夾路徑：**
```
C:\Users\...\桌面\site11  ← 任何名稱都可以！
```
