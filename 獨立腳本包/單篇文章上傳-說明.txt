========================================
單篇文章上傳腳本 - 使用說明
========================================

🎯 功能說明
-----------
這個腳本可以：
- ✅ 選擇單個 HTML 文章檔案
- ✅ 自動判斷網站名稱
- ✅ 上傳到 Strapi
- ✅ 推送到 GitHub（只推送這個單個 HTML 檔案）

📋 重要說明
-----------
✅ 只處理單個 HTML 檔案
✅ 不會上傳整個資料夾
✅ 只推送單個檔案到 GitHub

🚀 使用方式
-----------

步驟 1：執行腳本
獨立腳本包\單篇文章上傳.bat

步驟 2：選擇文章檔案
請輸入文章檔案完整路徑: 
C:\Users\...\site6\articles\2025-12-07.html

步驟 3：自動處理
- 自動判斷網站名稱（site6）
- 上傳到 Strapi
- 推送到 GitHub（只推送這個 HTML 檔案）

📝 範例
-----------

範例 1：上傳單篇文章
1. 執行腳本
2. 輸入：C:\Users\...\site6\articles\2025-12-07.html
3. 完成！

範例 2：上傳多篇文章
可以多次執行腳本，每次選擇不同的文章檔案

💡 注意事項
-----------
- 檔案必須是 HTML 格式
- 檔案必須在 Git 倉庫中（或父目錄在 Git 倉庫中）
- 只會推送單個 HTML 檔案，不會推送整個資料夾
