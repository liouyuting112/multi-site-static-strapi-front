## 💾 設定檔

### 設定檔位置

設定會自動儲存到：
```
獨立腳本包/上傳設定.txt
```

### 設定檔內容

```
GITHUB_URL=https://github.com/liouyuting112/multi-site-static-strapi-front
GITHUB_BRANCH=main
STRAPI_URL=https://effortless-whisper-83765d99df.strapiapp.com
STRAPI_TOKEN=您的Token
```

### 其他電腦使用

1. 複製 `上傳設定.txt` 到其他電腦
2. 執行腳本
3. 選擇使用現有設定
4. 完成！
