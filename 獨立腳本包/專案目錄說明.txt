========================================
專案目錄路徑說明
========================================

❓ 什麼是「專案目錄」？

專案目錄是您本機電腦上已經 Clone 下來的 GitHub 倉庫資料夾。

========================================
如何找到專案目錄？
========================================

方法 1：如果您已經 Clone 過

1. 打開檔案總管
2. 找到您之前 Clone 的資料夾
   通常位置：
   - C:\Users\您的名稱\Documents\multi-site-static-strapi-front
   - C:\Users\您的名稱\Desktop\multi-site-static-strapi-front
   - 或其他您選擇的位置

3. 複製完整路徑
   - 在檔案總管中，點擊地址欄
   - 或右鍵資料夾 → 內容 → 複製位置

方法 2：如果沒有 Clone 過

選擇 N（沒有專案目錄），讓腳本自動 Clone

========================================
正確的路徑格式
========================================

✅ 正確範例：
C:\Users\YourName\Documents\multi-site-static-strapi-front
C:\Users\YourName\Desktop\multi-site-static-strapi-front
D:\Projects\multi-site-static-strapi-front

❌ 錯誤範例：
https://github.com/liouyuting112/multi-site-static-strapi-front
liouyuting112/multi-site-static-strapi-front
multi-site-static-strapi-front

========================================
快速檢查

1. 路徑應該以磁碟機代號開頭（C:\ 或 D:\）
2. 路徑應該指向一個資料夾，不是檔案
3. 資料夾中應該有 .git 資料夾（表示是 Git 倉庫）

========================================
如果找不到專案目錄

建議選擇 N（沒有專案目錄），讓腳本自動 Clone：
- 腳本會自動下載倉庫
- 您只需要選擇要放在哪裡
- 預設位置：C:\Users\您的名稱\Documents\multi-site-static-strapi-front
