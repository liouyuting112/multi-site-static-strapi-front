# 🔍 智能搜索并编辑文章工具

## 🚀 使用方法

### 基本用法
```bash
npm run edit "搜索關鍵字"
```

或直接：
```bash
node search-edit.js "搜索關鍵字"
```

## 📝 搜索範例

### 1. 搜索文章內容片段
```bash
node search-edit.js "那天下午，窗外的陽光斜斜地照進來"
```

### 2. 搜索標題
```bash
node search-edit.js "瑪利歐"
node search-edit.js "為什麼現在的遊戲"
```

### 3. 搜索 Slug
```bash
node search-edit.js "retro-vs-modern"
node search-edit.js "2023-12-01"
```

### 4. 搜索描述
```bash
node search-edit.js "像素藝術"
```

## 🎯 工作流程

### 步驟 1：搜索
```bash
node search-edit.js "那天下午，窗外的陽光"
```

### 步驟 2：查看結果
工具會顯示所有匹配的文章：
```
🔍 搜索結果: "那天下午，窗外的陽光"
================================================================================

找到 1 篇匹配的文章:

1. 為什麼現在的遊戲沒有以前好玩？
   站點: site1 | 分類: fixed | Slug: retro-vs-modern
   匹配: 內容
   內容片段: ...那天下午，窗外的陽光斜斜地照進來，剛好打在我那台積了一點灰塵的 PS5 上。我盯著螢幕上的讀取條...
```

### 步驟 3：選擇編輯
- 如果只有一篇文章：直接詢問是否編輯
- 如果有多篇文章：輸入編號選擇

### 步驟 4：自動啟動編輯工具
選擇後，工具會自動啟動 `edit-article.js` 進行編輯。

## ✨ 功能特點

✅ **智能搜索** - 可以搜索標題、內容、描述、slug  
✅ **內容匹配** - 可以通過文章內容的任何片段找到文章  
✅ **匹配提示** - 顯示匹配的內容片段  
✅ **自動編輯** - 找到後自動啟動編輯工具  
✅ **多結果選擇** - 如果有多個結果，可以選擇要編輯的

## 📊 搜索優先級

搜索結果按匹配度排序：
1. **標題匹配** - 分數最高（10分）
2. **Slug 匹配** - 5分
3. **描述匹配** - 3分
4. **內容匹配** - 1分
