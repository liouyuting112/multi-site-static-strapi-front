# 🎯 按部分編輯文章工具

這個工具可以讓你**只編輯文章的某個特定部分**，而不需要處理整篇文章。

## 🚀 使用方法

```bash
node edit-parts.js edit site1 retro-vs-modern
```

或使用 npm：

```bash
npm run edit-part -- site1 retro-vs-modern
```

## 📋 工作流程

### 1. 啟動編輯

```bash
node edit-parts.js edit site1 retro-vs-modern
```

### 2. 查看文章結構

工具會自動解析文章，顯示結構：

```
📋 文章結構:

  1. [圖片] 圖片
     Alt: 左邊是像素遊戲畫面，右邊是現代3D高畫質遊戲畫面
     Src: https://github.com/.../site1-fixed1.webp?raw=true

  2. [段落] 那天下午，窗外的陽光斜斜地照進來，剛好打在我那台積了一點灰塵的 PS5 上...

  3. [段落] 這其實不是因為宮本茂覺得這樣比較帥...

  4. [標題] H2: 嘴巴太難畫，不如遮起來

  5. [段落] 在 1981 年《大金剛》這款遊戲開發時...
```

### 3. 選擇要編輯的部分

輸入編號，例如：
- `1` - 編輯第一張圖片
- `2` - 編輯第二個段落
- `4` - 編輯標題

### 4. 輸入新內容

根據選擇的部分類型：

**如果是圖片：**
- 輸入新的 Alt 描述
- 例如：`新的圖片描述文字`

**如果是段落：**
- 輸入新的段落文字
- 例如：`這是修改後的段落內容`

**如果是標題：**
- 輸入新的標題文字
- 例如：`新的標題`

### 5. 自動更新

工具會：
- 只更新選中的部分
- 保持其他內容不變
- 自動更新到 Strapi

## 💡 使用場景

### 場景 1：只修改圖片描述

```bash
node edit-parts.js edit site1 retro-vs-modern
# 選擇 1（圖片）
# 輸入新的 Alt 描述
# 完成！
```

### 場景 2：只修改某個段落

```bash
node edit-parts.js edit site1 retro-vs-modern
# 選擇 2（段落）
# 輸入新的段落文字
# 完成！
```

### 場景 3：查看完整內容

```bash
node edit-parts.js edit site1 retro-vs-modern
# 輸入 "all"
# 查看完整 HTML 內容
```

## ✨ 優點

✅ **精確編輯** - 只修改需要的部分  
✅ **快速操作** - 不需要處理整篇文章  
✅ **結構清晰** - 自動識別圖片、段落、標題  
✅ **安全更新** - 只更新選中的部分，其他內容不變

## 📊 支援的元素類型

- 📷 **圖片** (`<div class="hero-image">`)
- 📝 **段落** (`<p>`)
- 📌 **標題** (`<h2>` 到 `<h6>`)
- 📋 **列表** (`<ul>`, `<ol>`)

## ⚠️ 注意事項

1. **圖片編輯**：目前只支援修改 Alt 描述
2. **段落編輯**：會替換整個段落，保留 `<p>` 標籤
3. **標題編輯**：會保留標題等級（H2、H3 等）

## 🔄 與完整編輯的比較

| 功能 | 按部分編輯 | 完整編輯 |
|------|-----------|---------|
| 精確度 | ✅ 只改需要的部分 | ❌ 需要處理整篇 |
| 速度 | ✅ 快速 | ⚠️ 較慢 |
| 安全性 | ✅ 其他內容不變 | ⚠️ 可能誤改 |
| 複雜修改 | ❌ 不適合 | ✅ 適合 |

## 🎯 推薦使用場景

**使用按部分編輯：**
- 只修改圖片描述
- 只修改某個段落
- 只修改標題
- 快速小修改

**使用完整編輯：**
- 大範圍修改
- 重新組織結構
- 添加新內容
- 複雜的 HTML 修改
