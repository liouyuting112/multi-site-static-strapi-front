# Strapi 後台手動匯入指南

由於 SSL 連接問題，建議直接在 Strapi 後台手動建立內容。以下是詳細步驟：

## 📋 需要匯入的內容

### 1. 頁面（Pages）- 每個站點 4 個頁面

每個站點（site1 ~ site5）需要建立以下頁面：
- **home** (index.html) - 首頁
- **contact** (contact.html) - 聯絡我們
- **about** (about.html) - 關於我們
- **privacy** (privacy.html) - 隱私政策

**總共：5 個站點 × 4 個頁面 = 20 個頁面**

### 2. 固定文章（Posts - category: fixed）- 每個站點 3 篇

每個站點有 3 篇固定文章：
- site1: `retro-vs-modern`, `collector-guide`, `cartridge-care`
- site2-5: 類似結構

**總共：5 個站點 × 3 篇 = 15 篇固定文章**

### 3. 每日精選文章（Posts - category: daily）- 每個站點 3 篇

每個站點有 3 篇每日精選文章（最新的 3 篇）

**總共：5 個站點 × 3 篇 = 15 篇每日文章**

---

## 🚀 手動建立步驟

### 步驟 1：登入 Strapi 後台

訪問：`https://tidy-fireworks-ad201d981a.strapiapp.com/admin`

### 步驟 2：建立頁面（Pages）

1. 進入 **Content Manager** → **Page**
2. 點擊右上角 **"Create new entry"**
3. 填寫以下欄位：

   **範例：site1 的首頁**
   - **Site**: `site1`
   - **Type**: `home`（從下拉選單選擇）
   - **Slug**: `index`
   - **Title**: `懷舊時光機`（從 `site1/index.html` 的 `<title>` 標籤取得）
   - **HTML**: 從 `site1/index.html` 複製 `<main>` 標籤內的內容

4. 點擊右上角 **"Save"**
5. 點擊 **"Publish"**（發布）

6. 重複步驟 2-5，建立其他頁面：
   - site1: contact, about, privacy
   - site2-5: 所有 4 個頁面

### 步驟 3：建立固定文章（Posts）

1. 進入 **Content Manager** → **Post**
2. 點擊 **"Create new entry"**
3. 填寫以下欄位：

   **範例：site1 的固定文章**
   - **Site**: `site1`
   - **Category**: `fixed`（從下拉選單選擇）
   - **Slug**: `retro-vs-modern`（從檔案名稱取得）
   - **Title**: 從 `site1/articles/retro-vs-modern.html` 的 `<title>` 標籤取得
   - **HTML**: 從檔案中複製 `<article class="article-content">` 內的內容
   - **Date**: 可選，從檔案名稱或內容中取得
   - **Is Featured**: `false`（固定文章不需要）

4. 點擊 **"Save"** 和 **"Publish"**

5. 重複建立其他固定文章

### 步驟 4：建立每日精選文章（Posts）

步驟與固定文章相同，但：
- **Category**: `daily`
- **Is Featured**: `true`（每日精選文章需要標記為 featured）
- 只建立最新的 3 篇（根據日期）

## 💡 快速技巧

### 如何快速取得 HTML 內容

1. **頁面內容**：
   - 打開 `site1/index.html`
   - 找到 `<main>` 標籤
   - 複製 `<main>` 和 `</main>` 之間的所有內容

2. **文章內容**：
   - 打開 `site1/articles/retro-vs-modern.html`
   - 找到 `<article class="article-content">` 標籤
   - 複製標籤內的所有內容（不包括 `<h1>` 標題）

### 批量建立技巧

1. 一次建立一個站點的所有內容
2. 先建立所有頁面，再建立文章
3. 使用瀏覽器的「上一頁」功能，可以保留部分欄位內容

---

## ⚠️ 注意事項

1. **HTML 內容**：確保複製的內容是完整的 HTML，包括所有標籤
2. **編碼問題**：如果內容顯示亂碼，確保檔案是 UTF-8 編碼
3. **圖片連結**：檢查圖片 URL 是否正確
4. **發布狀態**：記得點擊 **"Publish"**，否則內容不會在前端顯示
