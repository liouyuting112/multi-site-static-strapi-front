# 圖片檢測工具使用說明

## 功能
自動檢測網站中使用的圖片，找出在GitHub倉庫中缺失的圖片，並生成包含中英文描述的Markdown清單供下載使用。

## 文件說明

### 核心文件
- `檢測缺失圖片並下載.cjs` - 主程序（Node.js）
- `執行檢測.bat` - 只執行檢測，生成報告和清單
- `一鍵檢測並下載.bat` - 檢測後自動調用Python程序下載圖片

### 生成的檔案
- `缺失圖片報告.txt` - 詳細報告
- `缺失圖片清單.md` - Markdown格式清單（包含中英文描述）
- `downloaded_images/` - 下載的圖片存放目錄

### Python抓圖程序
- `AI抓取無版權圖/` - 你的Python抓圖程序目錄

## 使用方式

### 方式一：只檢測
雙擊 `執行檢測.bat`，會生成：
- 缺失圖片報告.txt
- 缺失圖片清單.md（包含中英文描述）

### 方式二：檢測並下載
雙擊 `一鍵檢測並下載.bat`，會：
1. 檢測缺失的圖片
2. 自動調用Python程序下載圖片
3. 圖片保存到 `downloaded_images` 資料夾

## 描述格式

生成的Markdown清單包含：
- **描述（中文）**: 從HTML的alt屬性提取的中文描述
- **描述（英文）**: 根據圖片類型自動生成的英文關鍵詞
- **生成指令**: 中英文描述組合，可直接用於圖片生成

這樣可以確保無論使用中文還是英文搜尋，都能找到合適的圖片。
