## 幻灯片 5: 静态网页生成 - 三种方式

### 方式一：AI直接生成
- 快速原型开发
- 灵活性高

### 方式二：基于HTML模板
- 统一架构结构
- 批量生成

### 方式三：基于Figma设计稿
- 精确还原设计
- 品牌项目

---

## 幻灯片 6: 方式一 - AI直接生成

**流程：**
1. 在Cursor中提供需求描述
2. AI生成完整静态网页
3. 检查优化

**适用场景：**
- 全新网站
- 快速原型
- 概念验证

---

## 幻灯片 7: 方式二 - 基于模板生成

**流程：**
1. 准备HTML模板
2. 提供给Cursor
3. 说明主题和内容要求
4. AI按模板结构生成

**优势：**
- 保持统一架构
- 结构规范
- 易于维护

---

## 幻灯片 8: 方式三 - 基于设计稿生成

**流程：**
1. Figma设计完成
2. 导出设计图片
3. 提供给Cursor
4. AI按设计稿生成

**优势：**
- 设计还原度高
- 视觉精确
- 适合品牌项目

## 幻灯片 20: Vercel - 部署流程

### 首次部署步骤

1. **连接GitHub**
   - 登录Vercel
   - Add New Project
   - 授权GitHub

2. **选择项目**
   - 选择仓库
   - Import

3. **配置项目**
   - Framework: Other
   - Build Command: 留空
   - Output Directory: 留空

4. **部署**
   - 点击Deploy
   - 等待完成

## 幻灯片 27: 性能优化

   - 优化图片格式
   - 使用CDN

2. **优化生成方式**
   - 预生成静态页面
   - 增量更新

3. **缓存策略**
